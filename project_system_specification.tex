\section{Project Scope Alignment}

This document is rewritten to match the currently implemented project modules. The specification is aligned with the actual mobile prototype screens and flows present in the project codebase.

\subsection{Included Core Features}
\begin{itemize}
    \item Vet Registration
    \item Appointment Booking and Appointment Schedule
    \item Farm Management as My Animals and Add Animal
    \item Vaccination Management, including vaccine catalog, give vaccine flow, calendar, records, and vaccination centers
\end{itemize}

\subsection{Excluded from This Specification}
\begin{itemize}
    \item AI Disease Detection
    \item Payment System as a standalone feature
\end{itemize}

Although cancellation and reschedule fee warnings appear in the appointment module, they are treated here as appointment policy logic, not as a separate payment feature.

\section{Feature 2: Appointment Booking and Appointment Schedule}

\subsection{Sub Features}
\begin{itemize}
    \item Vet search by keyword
    \item Vet filtering by service type, specialty, availability, nearby option, and fee range
    \item Vet list with location, price, rating, and online status
    \item Transition from list to vet profile
    \item Personalized consultation mode selection from In Person, Video consultation, Chat, and Request vet to visit
    \item In Person booking through weekly time schedule rows
    \item Video consultation as instant video call when the veterinarian is online
    \item Video consultation as scheduled future slot when the veterinarian is offline
    \item Chat redirect for direct text conversation
    \item Request vet to visit slot selection
    \item Multi-animal selection for one appointment request
    \item Symptom note submission with the booking request
    \item Dynamic payment summary based on consultation mode, extra animals, travel fee, and added charges
    \item Appointment schedule viewing
    \item Accepted and pending appointment tracking
    \item Reschedule support
    \item Cancel support
    \item Extra payment warning for vet visit reschedule or cancellation when less than 24 hours remain
    \item Conflict prompt when the selected schedule overlaps with another appointment
    \item Focused appointment detail view when opened from the calendar
\end{itemize}

\subsection{Metadata}
\textbf{Vet Search and Discovery:} SearchTerm, ServiceType, Specialty, Availability, NearbyOnly, MinPrice, MaxPrice\\
\textbf{Vet Card Data:} VetID, VetName, SpecialtyName, ServiceTypes, Location, Price, Rating, AvailabilityLabel, OnlineStatus\\
\textbf{Consultation Option Metadata:} SelectedVisitMode, InPersonSchedule, VideoStatus, ChatRedirect, VetVisitSlot\\
\textbf{Booking Input Metadata:} SelectedDate, SelectedSlot, SelectedAnimalIDs, Symptoms, FarmerName\\
\textbf{Appointment Data:} AppointmentID, VetName, AnimalNames, AppointmentDate, AppointmentTime, Center, Mode, Status, AdditionalCharge\\
\textbf{Schedule Control:} ActiveSection, SelectedAppointmentID, Message, HoursRemaining, ShowConflictWarning\\
\textbf{Pricing Metadata:} BasePrice, ExtraAnimalCharge, TravelCharge, TotalPrice\\
\textbf{Policy Logic:} RescheduleChargeAmount, CancellationChargeAmount, ModeType, LateChangeRule, LessThan24HoursRule

\section{Feature 4: Vaccination Management}

\subsection{Sub Features}
\begin{itemize}
    \item Vaccine report summary card
    \item Vaccine catalog browsing
    \item Vaccine search
    \item Vaccine filtering by animal type, disease, age, weight, and teeth count
    \item Add vaccine catalog entry
    \item Give vaccine flow for selected animals
    \item Eligibility and timing check
    \item Vaccination table showing last date, current date, duration, and right-time or wrong-time status
    \item Vaccination schedule calendar
    \item Combined view of vaccine reminders and vet appointments
    \item Appointment-mode-aware labels inside the calendar such as In Person, Video call, and Vet visit
    \item Direct navigation from appointment cards to exact appointment details
    \item Vaccination records with pending to done confirmation
    \item Per-animal vaccination entry creation after confirmation
    \item Eligible and ineligible animal separation
    \item Vaccination center discovery
    \item Book vaccination visit linkage to vet booking
    \item Automatic append of completed vaccine entries into animal history
\end{itemize}

\subsection{Current Project Behavior Reflected in This Specification}
\begin{itemize}
    \item The appointment module includes both vet discovery and appointment schedule management.
    \item The farm module is centered on ``My Livestock'' and a multi-step ``Add Animal'' flow.
    \item The vaccination module is broader than a simple catalog; it includes scheduling, confirmation, center lookup, and linkage back into animal history.
    \item The current project intentionally excludes AI disease detection and a standalone payment workflow from this rewritten specification.
\end{itemize}
